# 图片上传

洋芋田富文本编辑器的图片像上传功能仅作为辅助工具，帮助用户完成将图片上传到受本软件支持的第三方服务商的行为，本身不提供资源存储和访问服务。用户需要自行注册或购买第三方图片存储服务，由此产生的费用和与第三方服务商的法律关系由用户自行处理，详细的使用协议和免责条款请参阅[法律信息](https://potatofield.cn/richtexteditor/terms#eula)。目前受支持的第三方存储服务商包括腾讯云 COS 和图仓，未来本软件将提供对更多第三方服务商的支持。

## 腾讯云 COS

[腾讯云 COS](https://cloud.tencent.com/document/product/436) 全称“腾讯云对象存储”，是腾讯旗下的网络资源存储服务。使用腾讯云 COS 的服务需要注册腾讯云账号并开通 COS 服务，**用户可以免费使用一定额度的资源，超出额度后会收取一定费用**。可以访问腾讯云 COS 产品的官方文档查看[免费额度](https://cloud.tencent.com/document/product/436/6240)和[产品定价](https://cloud.tencent.com/document/product/436/6240)，开通腾讯云 COS 和为洋芋田富文本编辑器配置腾讯云 COS 图像上传的步骤可以参阅我们的[指南](https://potatofield.cn/richtexteditor/docs#upload-tcloud)。

## 图仓

[图仓](https://tucang.cc/)是一家由个人开发者维护的免费聚合网络存储服务提供商。**用户可以免费注册并使用其提供的网络图片存储服务，但上传到图仓的图片可能无法同步到微信公众号等自媒体平台。** 如果使用本软件的目的不包括编辑和美化自媒体平台的富文本内容，图仓可以作为优质且免费的图片存储方案。图仓的使用说明可以参考[官方文档](http://doc.tucang.cc/project-1/)，也可以访问我们的[指南](https://potatofield.cn/richtexteditor/docs#upload-tucang)以了解配置洋芋田富文本编辑器配置图仓图片上传的步骤。
